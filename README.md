## Installation

1. React App
2. Tailwind v3
3. Google Font(Poppins): As used in figma sample file
4. Font Awesome(React): In sample figma prototype, there are several icons. Font awesome icons have been used as webpage icons. 
3. React-Router-Dom v6: To navigate through different webpages without loading each times. 

### Deployment Platform

* Netlify

### Instruction

1. Check out the live website link + code to assess.
2. Git clone the assignment using CMD in local machine, use command "npm install" to install all pachages and start the server using "npm start". Before all that the react should be globally installed in local machine.
